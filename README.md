## 🚀 Getting Started

### Prerequisites
- Node.js 20+ or Bun
- npm / pnpm / yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/NikolayBohdanov/nikoxyz-portfolio.git
cd nikoxyz-portfolio

# Install dependencies
npm install
# or
pnpm install

# Run development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📝 Content Updates

### Homepage Content
Edit `app/page.tsx` to update:
- Hero section
- About text
- Stats

### Work Experience
Edit `app/components/work.tsx`

### Skills
Edit `app/components/skills.tsx`

### Tools & Stack
Edit `app/components/stack.tsx`

### Contact Info
Edit `app/components/contact.tsx`

### Blog Posts
Add MDX files to `app/blog/posts/` directory.

---

## 🎨 Customization

### Theme Colors
Edit `app/global.css` for light/dark mode customizations.

### Metadata & SEO
Update `app/layout.tsx`:
- Page title
- Meta description
- Open Graph tags
- Twitter card

### Avatar
Replace `public/avatar.png` with your image (recommended: 256x256px or larger, square).

## 🚢 Deployment

### Deploy to Vercel (Recommended)

1. Push to GitHub
2. Import repository in Vercel
3. Deploy (automatic)

Every push to `main` triggers automatic redeployment.

### Environment Variables

No environment variables needed for basic setup.

For newsletter integration or other features, add `.env.local`:

```bash
# Example (if adding newsletter)
CONVERTKIT_API_KEY=your_api_key
```
